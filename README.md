# Sistema de Cadastro de Pessoas (Java)

## 📌 Sobre o Projeto
Este projeto é um sistema simples de cadastro de pessoas desenvolvido em *Java*, com execução via terminal. Ele permite realizar operações básicas como:

- Cadastrar pessoas
- Listar pessoas cadastradas
- Buscar pessoa pelo nome
- Mostrar quantidade de pessoas cadastradas
- Remover pessoa da lista

O sistema utiliza conceitos fundamentais de *Programação Orientada a Objetos (POO)*, como:
- Classes e objetos
- Encapsulamento (getters e setters)
- Construtores
- Listas com `ArrayList`

---

## 🎯 Objetivo
Este projeto tem finalidade *educacional*, sendo utilizado em sala de aula para que os alunos possam:

- Compreender a estrutura básica de um programa em Java
- Praticar conceitos de POO
- Entender o funcionamento de listas (`ArrayList`)
- Trabalhar com entrada de dados via `Scanner`
- Visualizar a organização de um projeto simples

Além disso, o projeto também é utilizado para introduzir o uso de:

- *Git* (controle de versão)
- *GitHub* (armazenamento e colaboração de código)

---

## 🛠️ Tecnologias Utilizadas
- Java
- NetBeans (IDE utilizada no desenvolvimento)
- Git
- GitHub

---

## 🚀 Próximos Passos
A evolução deste projeto será a implementação de uma *interface gráfica*, utilizando:

- *Java Swing*

Com isso, o sistema deixará de ser executado no terminal e passará a ter uma interface visual mais amigável para o usuário.

---

## 📚 Observação
Este projeto foi desenvolvido com foco didático. O código é intencionalmente simples para facilitar o entendimento dos alunos iniciantes.
